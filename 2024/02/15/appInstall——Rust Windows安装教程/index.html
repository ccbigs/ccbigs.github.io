Rust Windows安装教程

1.1 准备工作

1.1.1 安装MinGW、VS

安装Rust，首先确保你的电脑已安装MinGW，安装步骤可参考：C++操作系列(一)：MinGW环境安装与配置（无报错版）_mingw windows 安装-CSDN博客

VS的安装步骤可参考：C++操作系列(二)：VSCode安装和配置C++开发环境_vscode 安装c++库-CSDN博客

1.1.2 下载Rust

进入Rust官网下载界面：安装 Rust - Rust 程序设计语言 (rust-lang.org)

注：使用网页下载速度会比较慢，可以复制下载链接，使用迅雷秒下。
